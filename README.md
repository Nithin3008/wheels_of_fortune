### Wheels of Fortune
---
An e-commerce project for purchasing cars

## Techonlogies Used
---
*React Js(useReducer,useContext)
*React Router 6
*vanila css
*react-loader-spinner
*react-toastify
*mockbee server

## Features
---
1 .Home Page
2.Product Listing Page
3.Single Product image
4.Filters
    1.Price
    2.Category
    3.Ratings
5.Wishlisth Management
6.Cart Management
7.Search
8.Loading & Alerts
9.Authentication  
    1.Sign-up
    2.Login
    3.Logout
10.Address Management
11.Checkout
