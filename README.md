### Wheels of Fortune
---
An e-commerce project for purchasing cars

## Techonlogies Used
---
* React Js(useReducer,useContext)
* React Router 6
* vanila css
* react-loader-spinner
* react-toastify
* mockbee server

## Features
---
*  Home Page #1
*  Product Listing Page
*  Single Product image
*  Filters
     * Price
     * Category
     * Ratings
*  Wishlisth Management
*  Cart Management
*  Search
*  Loading & Alerts
*  Authentication  
    * Sign-up
    * Login
    * Logout
* Address Management
* Checkout
